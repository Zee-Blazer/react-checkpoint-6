# Movie App (simple React checkpoint)

Small React app that demonstrates:
- Components: `MovieCard`, `MovieList`, `Filter` (title & rating), and an Add Movie form
- React hooks: `useState`
- Add a movie and filter by title / minimum rating

Run locally:

1. cd movie-app
2. npm install
3. npm start

The app will open on http://localhost:3000 (or another free port).
